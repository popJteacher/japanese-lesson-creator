## 2. プロジェクト概要

### 作るもの
教師向けHTML自動生成Webツール。教師がファイル（文型データ）をアップロードすると、以下の4つの出力物が自動生成される。

| 出力物 | 用途 | 受け手 |
|---|---|---|
| スライドHTML | 授業中に画面に映す | 学習者 |
| 宿題HTML | 授業後に学習者が使う | 学習者 |
| アクティビティHTML | 活動中に使うタスクシート | 学習者 |
| 教案docx | 授業準備に使う | 教師 |

### データの3層構造
```
lesson_NN.json（課マスター）
  └── 「この課で何を教えるか」文型・語彙・例文・フロー

session_NNN.json（セッション）
  └── 「この授業でどの文型をどの順で教えるか」

activity_catalog.json（活動カタログ）
  └── 全課共通の活動定義（現在18種類）
```

### 使用教科書と方針
**「日本語の教え方ABC」を唯一の基準教科書として採用。**

理由：みんなの日本語・げんきが扱う文型をすべて網羅しているため。

判断：将来「みんなの日本語」「げんき」に完全移行する場合はこのシステムを使う意味がなくなる。よってABCベースで固定する。他教科書への移行が必要になった場合は `lesson_NN.json` の中身とT/Lスクリプトを差し替えるだけで対応可能（エンジン側は変えない）。

ABCの「教え方の例」を忠実に実装することがこのシステムの最大の差別化要因。

### マネタイズ方針
- 教師が課単位または月額で課金するSaaS形式を想定
- 教案docxの品質が商品の品質に直結する
- 「手順まで書いてある教案」が価値の核心

## 3. 確定済み：課マスター作成ルール全14項目

### A：必須ルール（これがないと課マスターが壊れる）

#### A-1　例文はPDF照合してから書く
- 教科書PDFをその課の分だけ読み通してから例文を書く
- 「先に草案を書いて後で照合」は禁止
- 照合済みであることを `originalSources` に記録する
- 背景：lesson_02でPDF再照合後に5件の修正が発生した失敗の反省

#### A-2　登場人物を勝手に作らない
- 教科書に登場する固有名詞（リンさん・アインシュタイン等）はそのまま使う
- それ以外の人物が必要な場合は `Aさん/Bさん/Cさん` 等の教科書記号を使う
- 独自キャラクター（田中先生・山田さん等）は作らない
- 背景：lesson_01で「田中世界観」を設計後に全削除した最大の失敗の反省

#### A-3　教科書の固有名詞は変えない
- 教科書に書いてある固有名詞はそのまま使う
- 変えた場合は必ず `originalSources.replacementNote` に理由を記録する

#### A-4　canDoは「〜できる」の形で1文型につき1文
- 形式：「〜できる」で統一
- 数：1文型につき1文のみ
- 良い例：「『これ・それ・あれ』で近くのものと遠くのものを指し示せる」
- 悪い例：「指示詞を使う。これ・それ・あれの違いがわかる」（2文・形式バラバラ）
- 理由：スライドのcanDoカードと直結するため形式が崩れると生成が壊れる

#### A-5　語彙導入の教具を必ず記録する
語彙リストの各語に `vocabType` を付与する。

| vocabType | 説明 | 代表課 |
|---|---|---|
| `実物` | 教師が実際のものを持ってくる | 第2課（時計・ペン等） |
| `語彙カード` | 1語1枚の絵カード | 第1課（人物・職業） |
| `場面絵` | 複数要素が入った絵・地図 | 第7課（公園マップ） |
| `対比絵` | 2つを並べた対比の絵 | 第6課（大小かばん） |

#### A-6　画像の役割（imageRole）を例文・語彙ごとに明記する
`imageRole` フィールドを必須とする。

| imageRole | 説明 |
|---|---|
| `vocab_person` | 役割不明の人物絵カード |
| `vocab_object` | もの絵カード |
| `scene` | 場面絵（複数要素を含む） |
| `contrast` | 対比絵（2つを並べる） |

背景：「医者の格好をした人の画像が表示されるが、それが誰かわからない」という問題を防ぐ。

---

### B：品質ルール（これがないと課によってバラつく）

#### B-5　語彙の記録のしかた（3点セット）
① JLPTレベル（N5〜N1）を書く
② 「この課が初出か・前の課で習ったか」を書く（`_inheritedFromLessonX`）
③ 絵カード化できない語（「わたし」「です」等）は理由を `_note` に書く

#### B-6　practiceTemplatesは最低2つ
- 1文型につき必ず2つ書く
- 標準構成：「肯定文テンプレート」＋「疑問文または否定文テンプレート」
- 形式：`＿＿＿` 形式の穴埋めで統一

#### B-7　materialNeeds[]の付与ルール
- `intro_activity` と `main_activity` には**必須**
- `pattern`・`example` はオプション
- `wrapUp`・`review`・`intro_slide` には付与しない（schema規則）

#### B-8　推奨時間の目安

| ステップの種類 | 目安の範囲 |
|---|---|
| 復習 | 3〜5分 |
| 導入アクティビティ | 5〜10分 |
| 文型1つ | 3〜7分 |
| メインアクティビティ | 10〜20分 |
| まとめ | 2〜5分 |

実際の時間は `session_NNN.json` で上書き可能。

#### B-9　活動例の必須フィールド
ABCの活動例を課マスターに取り込むとき、以下を必ず書く：

| フィールド | 内容 |
|---|---|
| 活動名 | 例：「忘れ物探し」 |
| 設定 | どんな場面か（1〜2文） |
| 準備物 | 何を何枚用意するか |
| 会話例 | 最低1往復 |
| fadingStage | 5段階のどこか（stage1〜5） |
| ABCとの対応 | activity_catalog.jsonのどのIDと対応するか |

---

### C：運用ルール（作業効率に関わる）

#### C-9　作業ステップの順序
この順番を必ず守る：

1. **Step 1a**：骨格（文型の一覧と構造のみ）
2. **Step 1b**：語彙リスト（PDFを読んでから）
3. **Step 1c**：例文（PDFと照合しながら）
4. **Step 1d**：materialNeeds（例文が確定してから）
5. **Step 1e**：アクティビティ選定（ABCの活動例とcatalogを照合）

例文より先にmaterialNeedsを書かない。アクティビティ選定は最後。

#### C-10　変更履歴の書き方
`_meta.changes` には「変更内容」と「変更理由」の両方を必ず書く。理由のないエントリは不完全とみなす。

#### C-11　バージョン管理の分離
- `formatVersion`：全課共通のスキーマ版数（現在 `2.7` が最新）
- `lessonVersion`：この課だけの改訂番号（`1.0` から始める）
- 両方を `_meta` に並べて書く

#### C-12　教案の手順はABC準拠で書く
- T/Lスクリプトを含む詳細な手順を教案docxに出力する
- 課マスターには `teachingHint`（1〜2文の要点）だけ書く
- 教案には「日本語の教え方ABC 第N課 教え方の例に準拠」と明記する

## 5. 確定済み：システム・課マスターの問題点と改善方針

### レベル1：第3課を作る前に解決すべき問題

#### 問題1：`activity`セクションの廃止
lesson_01.jsonの末尾にある `"activity": { "selectedId": null }` は旧設計の遺産。現在は `flow[].activityId` で管理する設計に移行済み。
**対応：** lesson_01・02から削除。第3課以降では作らない。

#### 問題2：フィールド名の統一
ABCには「注意」と「プラスα」という明確なラベルがあるが、課マスターでは `note`・`plusAlpha`・`_comment` が混在している。
**対応：** ABCの「注意」→ `cautionNote`、「プラスα」→ `plusAlpha` に統一。`note` は廃止。

#### 問題3：語彙情報の二重管理
`patterns[].vocabulary[]` と `flow[].materialNeeds[]` に同じ語彙情報が重複している。
**対応：** 語彙情報は `patterns[]` のみで管理。`flow[].materialNeeds[]` には「教具の種類」だけを書き、詳細は `patterns[]` を参照する設計にする。

#### 問題4：ABCの活動例とactivity_catalogの対応が未記録
ABCの活動例と `activity_catalog.json` の18種類のアクティビティとの対応がどこにも書いていない。
**対応：** 課マスター作成時に `ABCactivityRef` フィールドを追加してABC活動例名とcatalog IDの対応を記録する（C-9のStep 1eで実施）。

## 7. PDF精読全データ：第1〜10課の完全記録

### 第1課　名詞

**文型一覧**

| ID | 文型 | 板書形式 | JLPT |
|---|---|---|---|
| p1 | 〜は〜です（人物） | 〜は　〜です。 | N5 |
| p2 | 〜ですか（はい/いいえ） / だれ | 〜ですか。はい、〜です。／いいえ、〜じゃありません。 | N5 |
| p3 | 〜の〜（所属） | 〜の　〜です。 | N5 |

**語彙リスト**

p1・p2共通（7語）：先生、医者、会社員、学生、男の人、女の人、外国人
p3専用（5語）：会社、学校、銀行、大学、デパート

**教科書原典例文**

| no | 文型 | 例文 |
|---|---|---|
| 1 | p1 | 鈴木さんは先生です。 |
| 2-1 | p2 | リンさんは先生ですか。 |
| 2-2 | p2 | はい、先生です。／いいえ、先生じゃありません。 |
| 2-3 | p2 | だれですか。―キムさんです。 |
| 3-1 | p3 | Aさんは会社の会社員です。 |
| 3-2 | p3 | Bさんは学校の先生です。 |
| 3-3 | p3 | Aさんはさくら銀行の会社員です。 |
| 3-4 | p3 | リンさんはさくら大学の学生です。 |
| 3-5 | p3 | Aさんはさくらデパートの会社員です。 |

**語彙導入の教具**
- 人物・職業：語彙カード（絵カード）＋世界地図
- 所属機関：語彙カード（建物イラスト）

**導入手順の核心（教え方の例より）**

p1：人物絵カードを見せながら「鈴木さんです。鈴木さんは先生です」と提示→リピート→学習者の実名で置換練習
p2：同じ絵カードを使い「〜ですか。はい/いいえ」のQ&A形式で導入。学習者が知っている有名人の写真を使って「だれですか」を導入
p3：自分の胸を指して「わたし。これはわたしのかばんです」→学習者のものを指して「〜さんのかばんです」→所属機関絵カードと組み合わせて導入

**文法知識（教師向け）**
- A＝Bの関係になる「〜は〜です」に限定して初級前半は導入する（「父は会社です」のような例は出さない）
- 否定は「じゃありません」と「ではありません」の両方があることを教えてよい
- 過去形「でした」・普通体「だ」にはこの段階で触れない
- 「〜さんは？」の文末は上がるイントネーション（↗）に注意

**活動例：会話「はじめまして」**
- 設定：学習者2人ペアで自己紹介をする
- 準備物：簡単な名刺カード（やや大きめ）、会話モデルのスライドまたは絵カード
- 手順：教師が一人二役でモデルを示す→学習者2人ずつペアになって自分の情報で自己紹介を繰り返す。相手を次々と変えて練習させる
- 会話例：「はじめまして。〇〇です。〇〇人です。どうぞよろしく。」「〇〇さんは先生ですか。」「はい、教師です。〇〇さんは？」「わたしは学生です。東西大学の学生です。」
- fadingStage：stage4〜5（半統制産出〜自由産出）

**注意（ABCより）**
- 絵カードの人物については「だれですか」でよいが、クラスメートには「どなたですか」を使う
- 導入時は教師が質問のモデルを示すとき、すぐに答えを言わず学習者の反応（うなずき・首振り）を待ってから答える
- 学習者の中には教師をしている人もいるので「先生＝教師」であることを伝え、自己紹介では「先生」を使わないよう注意

**プラスα（ABCより）**
- 「〜から来ました」は自己紹介表現としてそのまま教える
- AもBも教師の場合は「わたしも教師です」という「も」の使い方を教える

| こ | これ | この | ここ | こちら | こんな | こう |

**語彙リスト（10語）**
時計、腕時計、ペン、鉛筆、ケータイ、本、雑誌、新聞、かばん、消しゴム

| 1-2 | p1（それ・相手側） | それはペンです。 |
| 1-3 | p1（あれ・遠い） | あれはABCビルです。 |
| 2-1 | p2（否定先行型） | いいえ、ケーキじゃありません。何ですか。―これは消しゴムです。 |

**「これ・それ・あれ」の領域対立（図で整理すること）**
- 図1：話し手と聞き手の領域が対立（これ↔それ）
- 図2：両者が同じ近い領域を共有（これ・これ）
- 図3：両者が同じ相手側領域を共有（それ・それ）
- 図4：両者が共通の遠い領域を指す（あれ・あれ）
- 初級段階では現場指示のみ教える（文脈指示は扱わない）

**語彙導入の教具**
- 身近なもの（時計・ペン等）：実物（時計は掛け時計と腕時計の両方用意）
- 遠くのもの：窓から見えるものまたはスライド

**導入手順の核心（教え方の例より）**

p1：まず身近なものの語彙を実物で十分に練習させてから「これ」を導入。次に教師が学習者の近くへ移動して「それ」を導入。「あれ」は窓から見えるものを指して導入
p2：一見何かわからないもの（ケーキに見える消しゴム等）を使って「何？」と聞きたくなる状況を自然に作る
p3：自分のかばんを指して「これはかばんです」→自分の胸を指して「わたし。これはわたしのかばんです」と導入
p4：4〜5人の学習者のペンを集めて「〇〇さんのペンはどれですか」と問いかける。「どれ」のときは全体を輪を描くように指す
p5：アインシュタイン等の世界的有名人の写真を使って「この人はだれですか」→「この人はアインシュタインです」
p6：少し離れたところからわざとどの写真を指しているかわからないように「あの人はだれですか」→近づいて1枚ずつ指して「この人ですか」→「どの人ですか」

**文法知識（教師向け）**
- 「こそあ」には現場指示と文脈指示があるが、初級では現場指示のみ教える
- 「この・その・あの」のあとには必ず名詞がつくことを板書で示す
- 「〜の〜」の「の」は所属（第1課）と所有（第2課）で意味が異なることに注意
- 選ぶ対象が2つの場合は「どちら」（会話では「どっち」）で質問することを教える

**活動例：落としもの返しごっこ**
- 設定：互いの持ち物を集め、「こ・そ・あ・ど」を使ってQ&Aしながら持ち主に返すタスク
- 準備物：学習者5人分のペン・鉛筆・鍵・時計等を袋に入れて混ぜる
- 手順：学習者同士が席を離れ歩き回りながら「すみません。これは〇〇さんのペンですか」と確認して返す
- 会話例：A「すみません。これはBさんのペンですか。」B「はい、そうです。わたしのです。どうも、ありがとう。」／B「いいえ、わたしのペンじゃありません。」A「じゃ、どれですか。」B「えーと、これです。このペンです。わたしのペンはこれです。」
- fadingStage：stage4〜5

### 第3課　動詞文その1

**ます形の活用表（板書用）**

|  | 肯定 | 否定 |

| 現在／未来 | 〜ます | 〜ません |
| 過去 | 〜ました | 〜ませんでした |

**語彙（導入動詞5つに限定）**
起きます、寝ます、働きます、休みます、勉強します

**時を表す言葉の分類（板書用）**

| 分類 | 例 |
|---|---|
| 「に」を伴うもの | 9時、10日、6月、20××年 |
| 「に」を伴わないもの | きのう、きょう、けさ、毎晩、毎日 |
| どちらでもいいもの | 昼、晩、火曜日 |

**時制チャート（板書用）**

| きのう | きょう | あした |
|---|---|---|
| きのうのあさ　ゆうべ | けさ　こんばん | あしたのあさ |
| 〜ました／〜ませんでした | 〜ます／〜ません |  |

**語彙導入の教具**
動作絵カード（起きる・寝る・働く・休む・勉強するの5枚）＋時計

**導入手順の核心（教え方の例より）**

p1：人物絵カードを示して「木村さんは7時に起きます」→板書で助詞「に」に注目させる→教師自身の時間を言って学習者に聞く→学習者同士でQ&A
p2：ホワイトボードに「9:00〜12:00」と書いたり時計を使って「9時から12時まで働きます」→「ません」は左右に首を振りながら導入。「〜ます」「〜ません」と板書して形を印象づける
p3：カレンダーで曜日や「きのう・きょう・あした」を復習→過去の話をさせる。「はい、きのう勉強しました」のように長く答える学習者には短く答えるほうが自然と教える
p4：「きょう」「きのう」「8時」「11日に」などを並べ、「に」がつくかを判断させるスライドまたはワークシートを使う

**文法知識（教師向け）**
- 日本語の動詞は人称が変わっても語形変化しない（英語と違う点を自然に印象づける）
- 「〜ません」の否定形も機械的に教えるのではなく意味のある会話のやりとりの中で教える
- 「それから」などの接続詞は特に説明せず、教師が自然に使うのを聞かせる
- 「あしたの朝、5時に起きます」のように未来のことも「ます形」で言えることを示す

**活動例：インタビュー「何時に起きますか」**
- 設定：学習者が互いに起きる時間・寝る時間・勉強するかどうかを聞き合い、タスクシートに書き込む
- 準備物：教師が事前に作成したタスクシート（行：起きる・寝る・勉強する・その他、列：学習者名）
- 手順：教師が学習者1人とモデルを示す→学習者同士でインタビューを続け、シートに記入する
- 会話例：T「ちょっと、すみません。Aさんは毎朝、何時に起きますか。」A「7時に起きます。」T「ああ、そうですか。じゃ、何時に寝ますか。」A「11時に寝ます。」T「あ、11時ですね。ありがとうございました。」
- fadingStage：stage4

**プラスα（ABCより）**
- 「7時ごろ」のように「ごろ・ごろ」を教えてもよい
- タスクシートの質問欄に空欄を入れておき、学習者が自分で考えた質問を書き込めるようにする

**語彙（導入動詞）**

**場所を表す助詞の整理（教師向け）**

| 用法 | 助詞 | 例 |
|---|---|---|
| 存在の場所 | に | 〜にある／いる |

**導入手順の核心（教え方の例より）**

p1：既習動詞でウォーミングアップQ&A→絵カードで目的語となる名詞を導入→「パンを　食べます。パンを食べます」のように板書の助詞「を」に注目させる

p3：場所絵カードと動詞絵カードを組み合わせて文を作る。「ミラーさんは会社で昼ごはんを食べます」→板書「〜で〜を〜ます」

**文法知識（教師向け）**
- 「すしを食べますか」に対して「いいえ、すしを食べません」と答えるのは不自然。「いいえ、食べません」のように短く答えるよう指導する
- 「〜も」は「バナナも食べました」のように自然に使えることを示す
- 助詞を強調しすぎると不自然な言い方になるので注意

- 「ビールとワイン」のように「〔名詞〕と〔名詞〕」の言い方を教える
- 「ビールかワイン」のような「〔名詞〕か〔名詞〕」の言い方を教えてもよい
- タスクシートに絵を使って記入時間を短縮し、口頭練習を促進する

### 第5課　動詞文その3

**文型一覧**

| ID | 文型 | 板書形式 | JLPT |
|---|---|---|---|
| p1 | 〜へ行きます | 〜へ　いきます。 | N5 |
| p2 | 〜で来ます（交通手段） | 〜で　〜へ　きます。 | N5 |
| p3 | いつ（来ましたか） | いつ　〜へ　きましたか。 | N5 |
| p4 | 〜へ〜に行きます（目的） | 〜へ　〜に　いきました。 | N5 |
| p5 | 〜ませんか／〜ましょう | 〜ませんか。―いいですね。行きましょう。 | N5 |

**「行く」と「来る」の視点の違い（図で整理すること）**

- 話し手がAにいてBへ移動する場合→「Bへ行く」
- 話し手がBにいてAからBへ移動してくる場合→「Bへ来る」
- 「帰る」は話し手がいつもいる場所（自分の家など）に戻る動作

**語彙導入の教具**
場所絵カード（会社・学校・京都等）、乗り物絵カード（電車・バス・自転車・歩いて）

**導入手順の核心（教え方の例より）**

p1：ミラーさんの絵カードを見せて「毎日、会社へ行きます」→土曜日・日曜日の否定で「会社へ行きません」→京都・大阪等の場所カードに替えて練習
p2：「行きます」と「来ます」の意味の違いをしっかり確認してから交通手段を導入。視点マークを使って話し手の位置を視覚的に示す。「歩いて」もここで教える
p3：比較的最近来日した学習者を選んで「何月に日本へ来ましたか」→ホワイトボードに来日時期を書き「?」マークで隠して「いつ来ましたか」を導入
p4：「図書館」「本を読みました」「行きました」を板書→「アニカさんはきのう図書館へ本を読みに行きました」の形に統合。「読みました」の「ました」を線で消して「に」と書き直して視覚的に示す
p5：「〜ませんか」「〜ましょう」はセットで会話またはタスクで練習

**文法知識（教師向け）**
- 移動動詞に「へ/に」（目的地）と「で」（移動経路）と「を」（移動の場所）が伴う点で混乱しやすい。数をしぼって練習し文型を定着させてから発展させる
- 「行きます」の語頭の"i"は母音の無声化によって弱く発音される。外国人には「来ます」との聞き分けが難しい場合がある
- 学校などの対面授業では今学校にいるので「学校へ行きます」と言わないよう注意する
- 「勉強します」「食事します」などは「勉強」「食事」という名詞形で目的を表すこともできる。その場合は「の」に変わる（「日本語の勉強に来ました」）

**活動例：映画を見に行きませんか**
- 設定：ペアになって互いのスケジュールを聞きながら、二人でいつどこへ行くかを相談するタスク
- 準備物：1週間のスケジュールが書き込める手帳シート（各学習者に1枚）
- 手順：各自がシートに1週間の予定を書き込む→「〜に行きませんか」「いつがいいですか」「〜曜日はちょっと……」などの表現を使ってペアで相談→決まった予定をペアごとに発表
- 会話例：A「Bさん、今度一緒に映画を見に行きませんか。」B「いいですね。いつがいいですか。」A「来週の金曜日はどうですか。」B「来週ですか。来週の金曜日はちょっと……。土曜日はどうですか。」A「あ、いいですよ。じゃ、来週の土曜日に行きましょう。」
- fadingStage：stage4〜5

**注意（ABCより）**
- 「先生、あした先生のうちへ来てもいいですか」という誤用がよく見られる（母語の「来る」をそのまま日本語に訳した例）。母語の構造や文法を知っておくことが大切

### 第6課　形容詞文

**文型一覧**

| p1 | い形容詞＋名詞 | 大きいかばんです。 | N5 |
| p2 | な形容詞＋名詞 | 親切な人です。 | N5 |
| p3 | い形容詞です／くないです | 〜は〜です。／〜は〜くないです。 | N5 |
| p4 | な形容詞です／じゃありません | 〜は〜です。／〜は〜じゃありません。 | N5 |
| p5 | どんな〜ですか | どんな〜ですか。 | N5 |
| p6 | どうですか | 〜はどうですか。 | N5 |
| p7 | い形容詞の過去形 | 〜かったです。 | N5 |
| p8 | い形容詞くて／な形容詞で〜 | 〜は〜くて、〜です。／〜は〜で、〜です。 | N5 |
| p9 | 〜ですが、〜です | 〜は〜ですが、〜です。 | N5 |

**形容詞の活用表（板書用）**

| | 現在肯定 | 現在否定 | 過去肯定 | 過去否定 |
|---|---|---|---|---|
| い形容詞 | 大きいです | 大きくないです／大きくありません | 大きかったです | 大きくなかったです／大きくありませんでした |
| な形容詞 | 元気です | 元気じゃないです／元気じゃありません | 元気でした | 元気じゃなかったです／元気じゃありませんでした |

**語彙（代表的なもの）**
い形容詞：大きい/小さい、長い/短い、高い/低い、高い/安い、新しい/古い、重い/軽い、よい/悪い、白い・黒い・赤い・青い・黄色い
な形容詞：きれいな・静かな・元気な・親切な・ハンサムな

**語彙導入の教具**
対比実物（大きいかばんと小さいかばん、高い時計と安い時計）、対比絵カード

**導入手順の核心（教え方の例より）**

p1：大小2つのかばんを並べて「かばんです→大きいかばんです」と対比で導入。基本語彙は対比絵カードで導入（長い/短い、高い/安い等）
p2：お年寄りの荷物を持っている人の絵カードを見せて「親切な人です」を導入。「きれい」は「い」で終わるが「な形容詞」であることを特に注意して示す
p3：ダイヤモンド入りの高級時計と安い実用的な時計の絵を見せて「高いです」「安いです」→学習者の「いいえ、安いです」という答えを引き取って否定形を導入
p4：きれいに片付いた部屋の絵と散らかった部屋の絵を対比して「きれいです」「きれいじゃありません」を導入
p8：「OO店のカレーはどうですか」で学習者にいろいろな形容詞を言わせ板書→好ましさが反対にならないものを選んで「おいしくて、安いです」と導入。「い形容詞」は「くて」、「な形容詞」は「で」になることを教える
p9：「日本語はむずかしいですが、おもしろいです」のように好ましさが反対になる組み合わせで「〜ですが」を導入

**文法知識（教師向け）**
- 「い形容詞」と「な形容詞」は混乱を招かないよう区別して導入する。板書は右側を「い形容詞」、左側を「な形容詞」のように位置を決めると記憶に残りやすい
- 否定形は2つの形があるが、初めはどちらかに統一して提出する
- 「よい」の活用は例外（「いい」はあるが「いくない、いくなかった」の形はない）。はっきり提示する
- 「暖かい」「涼しい」などそれ自体好ましいイメージを持つものと、「暑い」「寒い」などどちらかというと好ましくないイメージを持つものがある。これは「くて/で〜です」の接続のしかたにも関わる
- 言葉の練習のために自分の国や家族について言わせるときは、プライベートなことは言いたくない人もいるので押しつけにならないよう注意

**活動例：忘れ物探し**
- 設定：学校に忘れ物をした学生が受付で尋ねるというタスク
- 準備物：色や形が違う傘・ノート・コートのイラストが書いてある絵カードを2セット。1セットはホワイトボードに貼る。もう1セットは学習者に1枚ずつ配る
- 手順：教師が受付役になって1人の学習者とモデルを示す→その学習者が次の受付役になって同様に続ける
- 会話例：A「あのう、すみません。」T「どうしましたか。」A「きのう、教室に傘を忘れました。」T「そうですか。どんな傘ですか。」A「白くて大きい傘です。」T「ああ、これですか。」A「はい、そうです。ありがとうございました。」
- fadingStage：stage4〜5

**プラスα（ABCより）**
- 「とても〜です」「あまり〜ないです」などの副詞の使い方も教える
- 「頭/歯が痛いです」などの病気や体調に関する表現も練習するとよい

### 第7課　存在

**文型一覧**

| ID | 文型 | 板書形式 | JLPT |
|---|---|---|---|
| p1 | （場所）に〜があります／います | 〜に　〜が　あります。 | N5 |
| p2 | 〜の（位置詞）に | 〜の　上/下/中…に　〜があります。 | N5 |
| p3 | 何かあります／何もありません | 〜に　何か　ありますか。／何も　ありません。 | N5 |
| p4 | 〜が〇つあります（助数詞） | 〜に　〜が　〇つ　あります。 | N5 |
| p5 | 〜は（場所）にあります／います | 〜は　〜に　あります。 | N5 |

**「ある」と「いる」の使い分け**
- 「いる」：人や動物（自ら動くもの）
- 「ある」：それ以外のもの（自ら動かないもの）
- 特に注意：生物でも動かない花や木には「ある」を使う

**伝えたい情報によって語順が変わる**
- 伝えたいのが「存在する人や物」→「〜に〜が〜」
- 伝えたいのが「存在する場所」→「〜は〜に〜」

**位置詞リスト**
上、下、中、近く、前、後ろ（初級では6語程度にしぼる）

**助数詞の代表（初級向け）**
〜つ（一般）、〜人（人）、〜本（細長いもの）、〜枚（薄いもの）、〜階（階数）

**語彙導入の教具**
公園のイラストマップ（複数のものが配置されている場面絵）、箱（透明・不透明）、実物

**導入手順の核心（教え方の例より）**

p1：公園のイラストマップを見せて「公園です。池を指して、池です。池があります。公園に池があります」→「いる」は人・動物の絵をマップに貼りながら「公園に子どもがいます」と導入
p2：机と本を使って「机の……上に……本が……あります」とジェスチャーをつけながら導入。位置詞は体を動かしながら練習すると楽しく覚えられる
p3：透明な箱・不透明な箱・空の箱の3つを使って「何かありますか」→「ありません」→「何もありません」の流れで導入。「何か」と「何が」の聞き分けが難しいことに注意
p4：箱の中にあめを7つ入れて「ひとつ、ふたつ……」と一緒に数えながら「箱の中にあめが7つあります」
p5：教室の絵カードを見ながら「時計はどこにありますか。時計は　ここにあります」→場所が答えの形で導入

**文法知識（教師向け）**
- 名詞文の否定からの類推で「ここにベンチではありません」のような誤用が出るので注意
- 「本があります」は机の上に1冊置いた場合も数冊置いた場合も言える（単数・複数にかかわらず使える）
- 「ある」には存在・所有・開催など複数の用法がある。初級では存在と所有のみ扱う

**活動例：どこにありますか・どこにいますか**
- 設定：インフォメーションギャップのある室内の様子を表す2枚の絵を使い、人やものの所在を問うタスク
- 準備物：オフィスの見取り図で人やものがすべて書き込んである図1と、一部が抜けている図2の2種類。質問する人Aは図2を持ち、答える人Bは図1を持つ
- 手順：Aは図2の欄外にあるものについて、その位置をBに聞いて図に書き込む
- 会話例：A「田中さんはどこにいますか。」B「山田さんの前にいます。」A「電話はどこにありますか。」B「山田さんの机の上にあります。」A「ああ、そうですか。」
- fadingStage：stage3〜4

### 第8課　〜は〜が文

**文型一覧**

| ID | 文型 | 板書形式 | JLPT |
|---|---|---|---|
| p1 | 〜は〜が〜です（外見・特徴） | リンさんは　かみが　ながいです。 | N5 |
| p2 | 〜は〜が好きです | わたしは　〜が　すきです。 | N5 |
| p3 | 〜は〜があります（所有） | キムさんは　くるまが　あります。 | N5 |
| p4 | 〜は〜がほしいです | わたしは　〜が　ほしいです。 | N5 |
| p5 | 〜を/が〜たいです | わたしは　〜を/が　みたいです。 | N5 |

**語彙（代表）**
体の部位：髪、目、鼻、耳、口、手、足、背
スポーツ：テニス、野球、サッカー、水泳、スキー
食べ物・飲み物：りんご、みかん、ジュース、コーヒー等

**語彙導入の教具**
人物外見絵カード（髪の長い人・短い人等）、所有物絵カード（車・家・ピアノ等）

**導入手順の核心（教え方の例より）**

p1：髪の長い人物の絵カードを見せて「リンさんです。長いですね。リンさんは髪が長いです」→同様に他の特徴でも練習。動物の特徴を使っても楽しく練習できる
p2：「先週の土曜日にテニスをしました。来週も友達とテニスをします……わたしはテニスが好きです」のように自分の話から自然に導入。学習者の好みを聞きながら練習
p3：車の前に立つ人の絵と、一戸建ての家・ピアノの絵カードを使って「キムさんです。これはキムさんの車です。キムさんは車があります」
p4：「〜があります」の練習後「わたしは車がありません。わたしはドライブが好きです。わたしは車がほしいです」と自然な流れで導入。第三者には「ほしがっています」となる点に注意
p5：「わたしも映画が好きです。今度の日曜日に、この映画を見たいです」と映画の広告を見せながら導入。動詞カードを使って「〜ます→〜たいです」の変換練習（フラッシュカード）

**文法知識（教師向け）**
- 「ほしい」「〜たい」は感情形容詞で、第三者には直接使えない（「〜ほしがっている」「〜たがっている」になる）
- 「〜たい」の対象を表す助詞はテキストによって「が」としているものと「を」としているものがあるので事前に確認する
- 自分のことについて「上手」とは言わないので「得意」を教える
- 「きらい」は少し強い言い方なので「苦手」を教えてもよい

**活動例：友達になりましょう**
- 設定：好みや趣味の合う友達を探すというタスク
- 準備物：好みや趣味について選択式になっているタスクシート（7番は自由記入欄）
- 手順：各自がシートに自分の好みや趣味を記入→3人ほどの相手に次々と聞いていき、好みや趣味の合う人を探す→好みが合う友達を見つけたペアが発表する
- fadingStage：stage4〜5

### 第9課　比較

**文型一覧**

| ID | 文型 | 板書形式 | JLPT |
|---|---|---|---|
| p1 | 〜は〜より〜です | 〜は　〜より　〜です。 | N5 |
| p2 | 〜と〜とどちら（のほう）が〜ですか | 〜と〜と　どちら（のほう）が　〜ですか。 | N5 |
| p3 | 〜と〜と〜の中でどれがいちばん〜 | 〜と〜と〜の中で　どれが　いちばん〜ですか。 | N5 |
| p4 | 〜（の中）で〜がいちばん〜 | 〜（のなか）で　〜が　いちばん〜。 | N5 |

**語彙導入の教具**
年間の月別平均気温表、乗り物絵カード（タクシー・バス・新幹線等）、スポーツ絵カード、食べ物絵カード

**導入手順の核心（教え方の例より）**

p1：年間の月別平均気温表を指しながら「11月は13度です。12月は……8度ですね。12月は11月より寒いです」
p2：タクシーとバスの絵カードを見せて「ここから駅までタクシーで行きます。何分ぐらいかかりますか。バスはどうですか。タクシーとバスとどちら（のほう）が速いですか」→「タクシーのほうが速いです」
p3：豚肉・鶏肉・牛肉の絵カードと値段を使って「豚肉と鶏肉と牛肉の中で、どれがいちばん安いですか」。学習者の国の物価と比較させると話が弾む
p4：スポーツ絵カードを見せて「Aさんはどんなスポーツをしますか」→「テニスやスキー、それからサッカーもします」→「スポーツ（の中）で何がいちばん好きですか」

**文法知識（教師向け）**
- 「AはBよりも大きいです」「BよりAのほうが大きいです」の2通りの言い方ができる
- 3つ以上を比較するとき、具体的に列挙する場合は「どれ」、カテゴリを挙げる場合は「何/どこ/だれ/いつ」を使う
- 「どちらが好きですか」に対して「どちらも好きです」「どちらも好きじゃありません」という答え方も教える
- 宗教上の理由や嗜好で肉を扱いにくい場合は飲み物や果物の値段で代用する
- あまり個人的なことには触れないよう配慮が必要

**活動例：会話「一緒に旅行したいですね」**
- 設定：ペアになって旅行に行くという設定で、行き先や乗り物について相談するタスク
- 準備物：行き先や乗り物を空白にしたタスクシート（会話例の下線部を自分で選択）
- 手順：モデル会話を何回か聞かせる→学習者がペアになってタスクシートを使いながら会話する→プランが決まったらペアごとに発表
- 会話例：A「Bさん、春休みに一緒に旅行したいですね。」B「いいですね。Aさんはどこがいいですか。」A「わたしは北海道か沖縄へ行きたいです。」B「いいですね。わたしは北海道のほうがいいですね。沖縄は、去年行きました。」A「そうですか。じゃ、北海道へ行きましょう。」B「何で行きますか。Aさんは、飛行機と船と新幹線の中でどれがいちばんいいですか。」A「飛行機がいいですね。飛行機がいちばん速いです。」B「じゃあ、飛行機で行きましょう。」
- fadingStage：stage4〜5

### 第10課　授受

**文型一覧**

| ID | 文型 | 板書形式 | JLPT |
|---|---|---|---|
| p1 | 〜に〜をあげます | 〜は　〜に　〜を　あげました。 | N5 |
| p2 | 〜に/から〜をもらいます | 〜は　〜に/から　〜を　もらいました。 | N5 |
| p3 | 家族の呼称 | 佐藤さんのお父さん＝わたしの父 | N5 |
| p4 | 〜にくれます | 〜は　わたしに　〜を　くれました。 | N5 |

**授受動詞の整理（板書用）**

| | あげる方向 | もらう方向 |
|---|---|---|
| 主語 | 与える人 | 受ける人 |
| 制約 | 聞き手・第三者への「あげる」は制約なし | 主語は「わたし」側 |
| 「くれる」の制約 | 受ける人は「わたし」または「わたしグループ」のみ | − |

**待遇表現（ここでは教えない）**

| 目上 | 同輩・目下 | 目下 |
|---|---|---|
| さしあげる | あげる | やる |
| いただく | もらう | もらう |
| くださる | くれる | くれる |

**家族の呼称**
他人の家族に使う（尊称）：お父さん、お母さん、お兄さん、お姉さん、弟さん、妹さん
自分の家族に使う（謙称）：父、母、兄、姉、弟、妹

**語彙導入の教具**
プレゼントの箱・花・本・チョコレート等の実物または写真、家族の顔を描いた絵カード（家系図形式）

**導入手順の核心（教え方の例より）**

p1：実際にものを持ってやりとりしながら体で覚えさせる。誕生日を話題にして「もうすぐAさんの誕生日です。わたしはAさんにプレゼントをあげます」→学習者Bを前に呼んで「Bさんは何をあげますか」→B「花をあげます。どうぞ。誕生日おめでとうございます」→「BさんはAさんに花をあげました」。もらった本人（A）に「〜さんはわたしに〜をあげました」と言わせないよう注意
p2：「あげます」の直後に導入。教師がAの後ろに行き、Aの視点に立って「Aさんはプレゼントをもらいました。Aさんは……」と発話を促す。「あげます」と「もらいます」の関係をまとめたスライド（矢印と視点マーク）を使うと視覚的にわかりやすい
p3：家族の顔を描いた絵カードをホワイトボードに貼り「佐藤さんの家族です。佐藤さんのお父さん・お母さん・お兄さん・お姉さん・妹さん・弟さん」→わたしカードを「さとうさん」の上に貼って「わたしの家族です。わたしの父です。母です」
p4：時計を持って兄の絵カードから自分のほうに近づけながら「わたしは誕生日に兄に時計をもらいました。誕生日に兄はわたしに時計をくれました」。「くれます」は「わたしに」または「わたしグループの人に」しか言わないことを強調する。視点マークを使う

**文法知識（教師向け）**
- 「あげる」の主語は与える人。聞き手・第三者への「あげる」は制約がないが「AさんはわたしにあげましたⓇ」とは言わない
- 「もらう」は受ける側に立った言い方で主語は受ける人。「AさんはわたしにもらいましたⓇ」とは言わない
- 「くれる」は学習者にとって最もわかりにくい。ものが話し手の方向に向かって移動する場合だけに使い、受ける人は「わたし」か「わたしグループ」に限定
- 「〜にあげる」の「に」は対象を表し、「〜にもらう」の「に」は出どころを表すので機能が違う。機関が出どころの場合は「から」を使う

**活動例：プレゼント交換**
- 設定：カードに描いたものを実際にやりとりして、発表するタスク
- 準備物：はがき大の白紙カード（人数分）、タスクシート（誰が誰に何をあげたか記入する表）
- 手順：学習者を3人ずつグループに分ける→「きょうはパーティーです。グループの2人の友達にプレゼントをあげましょう。何をあげますか」と言って学習者に絵や文字でカードに書かせる→互いにカードをあげたり、もらったりする→タスクシートに記入→「AさんはBさんにワインをあげました」「BさんはAさんにワインをもらいました」のように発表
- fadingStage：stage4〜5
