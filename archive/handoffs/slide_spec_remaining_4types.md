**前提：表紙・intro_slide・main_activity・wrapUp は v16 handoff にて確定済み**

## 確定済み4typeとの整合ルール

- 見出し（`header.label`）は全スライド共通でセクション名を表示する
- 英語表示はトグルON/OFFで切り替える（`sentenceEn` / `en` フィールドを使用）
- ふりがな表示はトグルON/OFFで切り替える（`ruby_dictionary.js` を使用）
- `patternRef` で対象パターンを絞り込む（p1/p2/p3 単位で1スライド生成）

## ✅ `pattern`（文型）

### 見出し
```
「文型①」「文型②」「文型③」
```
patternId（p1/p2/p3）から番号を解決する。

### 本文構成

```
┌─────────────────────────────────────────┐
│ 文型①                                   │
├─────────────────────────────────────────┤
│                                          │
│  ┌──────────────────────────────────┐   │
│  │  [名前] は  [職業/国籍]  です。  │   │  ← パターンボックス
│  └──────────────────────────────────┘   │
│                                          │
│  ✓ 自分や他の人の名前・職業・国籍を     │
│    「〜は〜です」で紹介できる。          │  ← canDo
│                                          │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐  │
│  │ 先生  │ │会社員 │ │ 学生  │ │ 医者  │  │  ← 語彙カードグリッド
│  └──────┘ └──────┘ └──────┘ └──────┘  │
│  ┌──────┐ ┌──────┐ ┌──────┐           │
│  │日本人 │ │中国人 │ │韓国人 │ …        │
│  └──────┘ └──────┘ └──────┘           │
└─────────────────────────────────────────┘
```

**パターンボックスの表示ルール：**
- `pattern.label` を大きく中央表示（例：「〜は〜です」）
- 「は」「の」「ですか」などの助詞・語尾 → 強調色（design_tokens に従う）
- 「〜」スロット部分 → 薄いグレー背景のボックス

**語彙カードの表示ルール：**
- 対象語彙：`vocabulary.byPattern[patternRef]` から取得
- 1カードに：画像（imageId）＋ 語（word）＋ ふりがな（reading）
- 英語トグルONで：語の下に `en` を追加表示
- `shareVocabWith` が設定されている場合（p2はp1と共有）→ p1の語彙を再利用表示

**表示しない要素：**
- `practiceTemplates`（→ practice スライドで使用）
- `conversationPhrases`（→ wrapUp または教師ノートで使用）
- `cautionNote`（→ 教師ノートのみ）
- `plusAlpha`（→ 教師ノートのみ）

## ✅ `example`（例文）

### 見出し
```
「例文①」「例文②」「例文③」
```
patternId（p1/p2/p3）から番号を解決する。

### 本文構成

```
┌──────────────────────────────────────────┐
│ 例文①                                    │
├──────────────────────────────────────────┤
│  ┌────────────────────────────────────── ┐│
│  │[━━━━16:9画像━━━━] │ 鈴木さんは先生です。★ ││ ← アンカー例文
│  │（左38%が画像エリア） │ Suzuki-san is a teacher.  ││
│  └──────────────────────────────────────┘│
│  ┌────────┐┌────────┐┌────────┐┌────────┐│
│  │[16:9画像]││[16:9画像]││[16:9画像]││[16:9画像]││ ← その他例文（4列）
│  │リンさんは││キムさんは││鈴木さんは││リンさんは││
│  │大学生です││会社員です││日本人です││中国人です││
│  └────────┘└────────┘└────────┘└────────┘│
└──────────────────────────────────────────┘
```

**画像アスペクト比（master_prompt_design_guide_v2_4.py テンプレートC 根拠）：**
- 例文画像（imageRole: scene）→ **16:9（横長）**
  `"Composition: Wide 16:9 shot, 50mm standard lens equivalent"`
- 語彙・キャラクター画像（imageRole: vocab_*）→ **1:1（正方形）**（patternスライド・introスライドで使用）
- **サイズ統一は不要。** 用途が異なるため比率も異なる設計が正しい。

**カード表示ルール：**

| 種別 | isAnchor | カードレイアウト | 表示内容 |
|---|---|---|---|
| アンカー例文 | `true` | 横幅100%・左38%が16:9画像エリア・右側にテキスト | 画像 ＋ sentence ＋ sentenceEn（トグル） ＋ ★マーク |
| その他例文 | `false` | 4列グリッド・上部に16:9画像・下部にテキスト | 画像 ＋ sentence のみ（sentenceEnはトグルで） |

**取得元：**
- `patterns[patternRef].examples[]` をすべて表示（5件）
- `isAnchor: true` のカードを先頭・大サイズで表示
- 残り4件を2列グリッドで表示

**表示しない要素：**
- `originalSources`（内部管理用・表示不要）
- `_comment`（内部管理用・表示不要）

## ✅ `practice`（練習）

### 見出し
```
「練習①」「練習②」「練習③」
```
patternId（p1/p2/p3）から番号を解決する。

### 本文構成

```
┌─────────────────────────────────────────┐
│ 練習①                                   │
├─────────────────────────────────────────┤
│                                          │
│  ┌────────────────────────────────────┐ │
│  │                                    │ │
│  │  わたしは  ［＿＿＿＿＿＿］ です。 │ │  ← テンプレート1（大きく）
│  │                                    │ │
│  │  ヒント：（名前・職業・国籍）       │ │  ← hint（小さく）
│  └────────────────────────────────────┘ │
│                                          │
│  ┌────────────────────────────────────┐ │
│  │                                    │ │
│  │  ［＿＿＿＿＿＿］さんは            │ │  ← テンプレート2（大きく）
│  │  ［＿＿＿＿＿＿］ です。           │ │
│  │                                    │ │
│  │  ヒント：（人物紹介）              │ │
│  └────────────────────────────────────┘ │
│                                          │
└─────────────────────────────────────────┘
```

**テンプレート表示ルール：**
- `patterns[patternRef].practiceTemplates[]` を全件表示
- 「＿＿＿」部分 → 目立つ色の下線ボックス（例：青い下線）
- `hint` → テンプレートの下に小さく薄字で表示
- テンプレートが複数ある場合（p2は3件）→ 縦に並べる

**解答表示：なし**
- 解答はなし。教師が口頭でフィードバックする設計のため。
- 解答スペースは設けない（学習者が口頭で答える）

**件数確認（第1課）：**
| patternRef | practiceTemplates件数 |
|---|---|
| p1 | 2件 |
| p2 | 3件 |
| p3 | 2件 |
