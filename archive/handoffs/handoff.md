## プロジェクトの全体像

教師が `session_NNN.json` をアップロードすると
スライド HTML / 宿題 HTML / アクティビティ HTML / 教案 docx の
4 ファイルをダウンロードできる **完全静的 Web ツール**。

### 設計の核
「日本語の教え方 ABC」準拠の課マスター（lesson_NN.json）を
「文型の網羅集」として使う。教師は session で文型を組み合わせるだけ。

### データの 3 層構造
```
lesson_NN.json     ← 課マスター（ABC 準拠・文型・語彙・例文）
session_NNN.json   ← 授業設定（どの文型をどう教えるか）
activity_catalog.json ← 活動カタログ（全課共通・18 種類）
```

```json
"teach": [
  { "lessonNo": 1, "patternId": "p1", "isNew": true },
  { "lessonNo": 1, "patternId": "p2", "isNew": true },
  { "lessonNo": 1, "patternId": "p3", "isNew": true }
]
```
